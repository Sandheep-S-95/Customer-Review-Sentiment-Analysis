# 🚀 Customer-Review-Sentiment-Analysis (NLP)

## 🌟 Overview

Welcome to our cutting-edge Natural Language Processing (NLP) web application! 🎉 This powerful tool is designed to classify customer product reviews using state-of-the-art machine learning techniques. Built with Python and StreamLit, it's your go-to solution for understanding customer sentiment! 📊💬

## 🛠️ Technologies Used

- 🐍 Python
- 🌊 StreamLit
- 🐼 Pandas
- 🔢 NumPy
- 📊 Seaborn
- ☁️ WordCloud
- 🔤 NLTK (Natural Language Toolkit)
- 🚀 XGBoost
- ⚖️ SMOTE (Synthetic Minority Over-sampling Technique)
- 🍲 BeautifulSoup (for web scraping)

## ✨ Features

- 📝 Process and analyze customer product reviews
- 🧠 Utilize advanced NLP techniques for text classification
- 📈 Visualize data insights with Seaborn and WordCloud
- 🔍 Handle imbalanced datasets with SMOTE
- 🌐 Test the model with real web-scraped data

## 🚀 Getting Started

Let's get this NLP party started! 🎊

### 📥 Clone the Repository

```bash
git clone https://github.com/Sandheep-S-95/Customer-Review-Sentiment-Analysis.git
cd Customer-Review-Sentiment-Analysi
```

### 🔧 Set Up a Virtual Environment (Optional but Recommended)

```bash
python -m venv venv
source venv/bin/activate  # On Windows, use `venv\Scripts\activate`
```

### 📦 Install Dependencies

```bash
pip install -r requirements.txt
```

### 🏃‍♂️ Run the Application

```bash
streamlit run app.py
```

Voila! 🎉 Your NLP web app should now be running at `http://localhost:8501`.

## 📋 Requirements

Ensure you have the following libraries installed:

```
streamlit
pandas
numpy
seaborn
wordcloud
nltk
xgboost
imbalanced-learn  # for SMOTE
beautifulsoup4
```

You can install them all at once using the `requirements.txt` file as shown in the installation steps above.

## 🤓 Usage

1. 🌐 Launch the app using the command provided above.
2. 📤 Upload your customer review dataset or use the web scraping feature.
3. 🔧 Adjust any parameters if needed.
4. 🚀 Click "Analyze" and watch the magic happen!
5. 📊 Explore the visualizations and insights generated by the app.
